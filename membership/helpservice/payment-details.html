Create Subscription Payment

Member: Select or enter the name of the member making this subscription payment. This field is an auto-complete field, which will present a list of choices after the first few letter have been typed. Payment date: Select the date of the subscription payment. Subscription: Select the subscription code for the subscription receiving a payment. Payment amount: Enter the amount of the payment, without a currency symbol (eg 55.00). Notes: Enter any notes in this free-form field to provide additional information about the subscription. This field is multi-line and can be expanded by dragging the bottom margin. All information entered in this field is fully indexed and can be used in searches.
