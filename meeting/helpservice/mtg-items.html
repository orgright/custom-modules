Create Meeting Item

Item: Enter a name for this meeting item. The name will be rejected if a meeting item with this name already exists for this meeting.

Member: enter the name of the member (if any) responsible for this meeting item. This is an auto-complete field which will present a list of choices once a few letters have been typed.

Meeting: enter the name of the meeting that this item belongs to. This is an auto-complete field which will present a list of choices once a few letters have been typed.

Meeting item type: Select the type of meeting item from the drop-down list. The choices available on this list are managed by the administrator and should reflect the types of item that the organisation is interested in recording.

Completion date: enter a date for review, bring-up or completion of this item depending on its type.

Meeting item details: enter details of this meeting item.

Notes: enter any general notes about this agenda item.
