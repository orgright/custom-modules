Create Project Report

Report: Enter a name for the project report. The name will be rejected if a report with this name already exists for the specified project. Responsible person: Enter the name of a member making this report. Depending on the number of members, this will either be a drop-down list or an auto-complete field (see making choices). Project: Enter the name of the project. Depending on the number of current projects, this will either be a drop-down list or an auto-complete field (see making choices). Activity: Enter the name of the project activity if this report pertains to a specific activity. Depending on the number of current activities, this will either be a drop-down list or an auto-complete field (see making choices). Report date: Enter the date of the project report. Description: Enter details of the project report. This is a multi-line field which can be expanded by dragging the bottom margin. Notes: Enter any general notes about this project report.
