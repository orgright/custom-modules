Link Project Actvities

Project: enter the name of a project to work with. This is an auto-complete field which will present a list of choices once a few letters have been typed. Activity: enter the name fo the first activity in the link relationship. This is an auto-complete field which will present a list of choices once a few letters have been typed. Link between activities: Specify the type of link between the first activity and this one by chosing from the list. Possible choices are:
  • no link between 1st and 2nd activities
  • 1st starts after 2nd finishes
  • 1st finishes before 2nd starts
  • 1st starts after 2nd starts
  • 1st starts before 2nd starts
  • 1st finishes after 2nd finishes
  • 1st finishes before 2nd finishes
  • 1st finishes after 2nd starts
  • 1st starts before 2nd finishes
Note that the listed linkages are in mirror pairs that will apply if the order of first and second activities is reversed.
